What is PageShot?

PageShot is an experimental Addon for Firefox that lets you save and share permanent copies of anything on the web.

Why use PageShot?

Save anything

PageShot lets you save anything on the web, including pages that requires a login or content that changes or is personalized

Share with anyone

Once you've created a shot, anyone you give the link to can view it; never worry about logins or disappearing content.

By someone you trust

PageShot is built and hosted by Mozilla, so you know it's safe to use.

How does PageShot work?

Install the Firefox Addon.

Install the addon and you'll get a new button to create shots.
